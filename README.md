アプリ名　
「Welcome Pets」

アプリケーション概要
保健所に連れてかれてしまう動物、ペットショップで売れ残ってしまった動物、飼い主がなんらかの理由で飼育を維持できなくなってしまったなど、様々な理由で場所を失ってしまった動物を一時的に保護し、別の飼い主を見つけることを目的としたアプリです。

利用方法
業者、または個人が動物の写真を投稿し、それを閲覧した人がコメントなどを通じて交渉し、最終的には引き渡しを完了するというものです。

目指した課題解決
犬や猫などを繁殖させて、売り物にする人がいることは耳にしていました。その中には、誰にもお世話をしてもらえずなくなってしまう命も少なくありません。そんな不幸な目に遭ってしまうペットが少しでも減ってくれることを祈ってこのアプリを考えました。どうしてもペットを飼えなくなってしまう環境にいる人も存在すると思います。そんな課題を解決するためにも使用して欲しいと考えました。
